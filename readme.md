# Личная карточка

Данный репозиторий содержит код одностраничного сайта-визитки. Проект предназначен для краткого представления информации, навыков и контактов.

## Особенности
* Минималистичный дизайн и чистая структура
* CSS-анимации для улучшения пользовательского опыта при взаимодействии с элементами
* Адаптивная верстка для корректного отображения на мобильных устройствах и десктопах
